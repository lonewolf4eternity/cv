About this Project

Bike Sharing Project

Untuk proyek ini, saya menggunakan Python untuk mengimpor dataset yang berekstensi .csv, melakukan cleaning/pembersihan, menganalisa dan memvisualisasikan dataset dari Kaggle bernama Bike Sharing Dataset. Bike Sharing System adalah sebuah generasi baru dari rental sepeda tradisional, dimana seluruh proses dari keanggotaan, rental dan pengembalian kembali menjadi otomatis. Melalui sistem ini pengguna bisa dengan mudah menyewa sepeda dari posisi tertentu ke posisi lainnya.

Kedua tabel memiliki kolom yang sama kecuali hr pada tabel hour.csv yang tidak tersedia pada tabel day.csv.

Berikut keterangan mengenai kedua tabel:

  • instant: indeks catatan
  • dteday : tanggal
  • season : musim (1: musim semi, 2: musim panas, 3: musim gugur, 4: musim dingin)
  • yr : tahun (0: 2011, 1: 2012)
  • mnth : bulan (1 sampai 12)
  • hr : jam (0 hingga 23)
  • holiday : hari cuaca hari libur atau tidak (diambil dari [Tautan Web])
  • weekday : hari dalam seminggu
  • workingday : jika hari tersebut bukan akhir pekan atau hari libur maka nilainya 1, jika tidak maka nilainya 0.
  • weathersite :
    • Cerah, Sedikit awan, Berawan sebagian, Berawan sebagian
    • Kabut + Berawan, Kabut + Awan pecah, Kabut + Sedikit awan, Kabut
    • Salju Ringan, Hujan Ringan + Badai Petir + Awan berserakan, Hujan Ringan + Awan berserakan
    • Hujan Lebat + Butiran Es + Badai Petir + Kabut, Salju + Kabut
  • temp: Suhu yang dinormalisasi dalam Celcius. Nilai diperoleh melalui (t-t_min)/(t_max-t_min), t_min = -8, t_max = +39 (hanya dalam skala per jam)
  • atemp: Suhu perasaan yang dinormalisasi dalam Celcius. Nilai-nilai ini diperoleh melalui (t-t_min)/(t_max-t_min), t_min=-16, t_max=+50 (hanya dalam skala per jam)
  • hum: Kelembapan yang dinormalisasi. Nilai dibagi menjadi 100 (maksimum)
  • windspeed: Kecepatan angin yang dinormalisasi. Nilai dibagi menjadi 67 (maks)
  • casual: jumlah pengguna biasa
  • registered: jumlah pengguna terdaftar
  • cnt: jumlah total sepeda sewaan termasuk yang kasual dan terdaftar

Membuat Visual (Dashboard)

Untuk dashboard, saya menggunakan modul streamlit yang terdapat didalam Python untuk membuat beberapa bahan/chart.

Untuk menampilkan tingkat penyewaan sepeda pada setiap musim, saya menggunakan column char.

Sedangkan untuk Konstribusi penyewaan sepeda per jam, saya menggunakan line chart.

Dapat dilihat pada dashboard bahwa musim gugur memiliki penyewaan sepeda terbanyak dengan rata-rata sewa 5644 di ikuti oleh musim panas dengan rata-rata 4992 penyewa, lalu musim dingin dengan rata-rata 4728 penyewa dan terakhir musim semi dengan rata-rata 2604.
