About this Project

Maven Market Project

Untuk proyek ini, saya menggunakan Power BI untuk mengimpor dataset yang berekstensi .csv, melakukan cleaning/pembersihan, menganalisa dan memvisualisasikan dataset dari Maven Analytic bernama Maven Market. Maven Market adalah bisnis yang memiliki berbagai toko kelontong dengan berbagai ukuran di berbagai negara termasuk Amerika Serikat, Kanada dan Meksiko.

Dalam analisis dasbor ini saya akan mengulas wawasan yang saya dapatkan dari pembuatan dan laporan, beberapa proses berpikir yang berbeda yang saya gunakan untuk mencapai kesimpulan tersebut serta menawarkan beberapa solusi potensial. Proyek ini merupakan proyek bonus dari Kursus Power BI Desktop Untuk Business Intelligence dari Maven Analytics dan dapat ditemukan di Udemy.

Dataset

Seperti yang dinyatakan sebelumnya, Maven Market adalah bisnis yang memiliki toko kelontong di beberapa negara yang berbeda dan dengan berbagai ukuran dan profitabilitas.

Data mentah berasal dari file CSV yang kemudian dibawa ke Power BI menggunakan Get Data. Data tersebut kemudian ditransformasikan dan dibersihkan menjadi set yang lebih mudah digunakan dan kemudian ditutup dan diterapkan. Proses ini diulangi untuk setiap tabel / file CSV sampai semua 8 tabel telah dibawa dan ditransformasikan.

Proses pembersihan untuk set data ini cukup mudah, terutama membutuhkan promosi header, koreksi tipe data, dan pembuatan kolom tambahan yang akan membuat pekerjaan dengan set data lebih mudah di kemudian hari. Selain itu, set data mentah menyediakan 2 tabel untuk transaksi, satu tabel untuk setiap tahun (1997 dan 1998). Karena kedua tabel ini memiliki struktur tabel dan kolom yang sama, maka kedua tabel tersebut digabungkan ke dalam satu tabel berjudul "Data Transaksi" yang berisi kedua tahun tersebut. Saya kemudian melakukan verifikasi data dan memastikan tabel baru tersebut memiliki jumlah baris dan kolom yang benar.

DAX Functions

Setelah melakukan koneksi antar tabel, saya melakukan beberapa perhitungan menggunakan DAX Function dimana perhitungan tersebut menghitung beberapa KPI yang saya butuhkan untuk visualisasi data, seperti :

  • End of Month
    End of Month =
    EOMONTH('Calendar'[date] , 0)
  • Weekend
    Weekend =
    IF (
    OR (
    'Calendar' [Day Name] = "Saturday",
    'Calendar' [Day Name] = "Sunday"
    ), "Y", "N"
    )
  • Current Age
    Current Age =
    YEARFRAC(
    'Customers'[Birthday], TODAY(), 1
    )
  • House Number
    House Number =
    IF (
    ISBLANK (
    'Customers'[customer_address]
    ), BLANK (),
    LEFT (
    'Customers'[customer_address],
    SEARCH (
    " ",
    'Customers'[customer_address]
    ) - 1
    )
    )
  • Priority
    Priority =
    IF (
    AND (
    'Customers'[homeowner] = "Y",
    'Customers'[member_card] = "Golden"
    ), "High", "Standard"
    )
  • Short_Country
    Short_Country =
    UPPER (
    LEFT (
    'Customers'[customer_country], 3
    )
    )
  • Price_Tier
    Price_Tier =
    IF (
    'Products'[product_retail_price] > 3,
    "High",
    IF (
    'Products'[product_retail_price] > 1,
    "Mid",
    "Low"
    )
    )
  • Unique Product
    Unique Product =
    DISTINTCOUNT (
    'Products'[product_name]
    )
  • All Returns
    All Returns =
    CALCULATE (
    COUNT (
    'Returns_1997-1998'[quantity]
    ), ALL (
    'Returns_1997-1998'
    )
    )
  • Last Month Return
    Last Month Return =
    CALCULATE (
    [All Returns],
    FILTER (
    ALL ('Calendar'),
    YEAR (Calendar[Date]) = YEAR (MAX(Calendar[Date]) - 1) &&
    MONTH (Calendar[Date]) = MONTH (MAX(Calendar[Date]) - 1)
    )
    )
  • Quantity Returned
    Quantity Returned =
    SUM (
    'Returns_1997-1998'[quantity]
    )
  • Return Rate
    Return Rate =
    DIVIDE (
    'Returns_1997-1998'[Quantity Returned],
    'Transaction Data'[Quantity Sold],
    0
    )
  • Total Returns
    Total Returns =
    COUNT (
    'Returns_1997-1998'[product_id]
    )
  • Year_Since_Remodel
    Year_Since_Remodel =
    IF (
    NOT (
    ISBLANK (
    Stores[last_remodel_date]
    )
    ), YEARFRAC (
    Stores[last_remodel_date],
    TODAY()
    )
    )
  • % Weekend Transaction
    % Weekend Transaction =
    DIVIDE (
    [Weekend Transaction],
    [Total Transaction],
    0
    )
  • 60-Day Revenue
    60-Day Revenue =
    CALCULATE (
    [Total Revenue],
    FILTER (
    ALL ('Calendar'),
    'Calendar'[date] >= MAX ('Calendar'[date]) - 59 &&
    'Calendar'[date] <= MAX ('Calendar'[date])
    )
    )
  • All Transaction
    All Transaction =
    CALCULATE (
    COUNT ('Transaction Data'[quantity]),
    ALL ('Transaction Data')
    )
  • Last Month Profit
    Last Month Profit =
    CALCULATE (
    [Total Profit],
    FILTER (
    ALL ('Calendar')
    YEAR ('Calendar'[Date]) = YEAR (MAX('Calesndar'[Date]) - 1) &&
    MONTH ('Calendar'[Date]) = MONTH (MAX('Calesndar'[Date]) - 1)
    )
    )
  • Last Month Revenue
    Last Month Revenue =
    CALCULATE (
    [Total Revenue], DATEADD (
    'Calendar'[date],
    - 1,
    MONTH
    )
    )
  • Last Month Transaction
    Last Month Transaction =
    CALCULATE (
    [All Transaction],
    FILTER (
    ALL ('Calendar')
    YEAR ('Calendar'[Date]) = YEAR (MAX('Calesndar'[Date]) - 1) &&
    MONTH ('Calendar'[Date]) = MONTH (MAX('Calesndar'[Date]) - 1)
    )
    )
  • Profit Margin
    Profit Margin =
    DIVIDE (
    [Total Provit],
    [Total Revenue],
    0
    )
  • Quantity Sold
    Quantity Sold =
    SUM (
    'Transaction Data'[quantity]
    )
  • Revenue Target
    Revenue Target =
    [Last Month Revenue] * 1.05
  • Total Cost
    Total Cost =
    SUM (
    Products
    Products[product_cost] * [Quantity Sold]
    )
  • Total Profit
    Total Profit =
    [Total Revenue] - [Total Cost]
  • Total Revenue
    Total Revenue =
    SUMX {
    Products,
    Products[product_retail_price] * [Quantity Sold]
    }
  • Total Transactions
    Total Transaction =
    COUNT (
    'Transaction Data'[product_id]
    )
  • Weekend Transaction
    Weekend Transaction =
    CALCULATE (
    [Total Transaction],
    'Calendar'[Weekend] = "Y"
    )
  • YTD Revenue
    YTD Revenue =
    CALCULATE (
    [Total Revenue],
    FILTER (
    ALL('Calendar'), 'Calendar'[date] <= MAX ('Calendar'[date]) &&
    YEAR (MAX('Calendar'[date]))
    )
    )
